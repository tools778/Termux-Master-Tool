# Termux-Master-Tool - Requirements
# ─────────────────────────────────────
# This project is a pure Bash script.
# No external language dependencies required.
#
# Runtime Requirements (must be available in Termux):
#   - bash          >= 5.0
#   - pkg           (Termux package manager)
#   - apt / apt-get (underlying package manager)
#   - ping          (for internet check)
#   - dpkg          (for package status queries)
#
# All of the above are pre-installed in Termux by default.
#
# Optional (enhances functionality):
#   - termux-api    (for battery level detection)
#
# Install termux-api:
#   pkg install termux-api -y
